# 基本计算器

## 题目

给你一个字符串表达式 s ，请你实现一个基本计算器来计算并返回它的值。

## 示例

### 1

```
输入：s = "1 + 1"
输出：2
```

### 2

```
输入：s = " 2-1 + 2 "
输出：3
```

### 3

```
输入：s = "(1+(4+5+2)-3)+(6+8)"
输出：23
```

## 提示

* `1 <= s.length <= 3 * 105`
* `s`由数字、`'+'`、`'-'`、`'('、')'`、和`' '`组成
* `s`表示一个有效的表达式
